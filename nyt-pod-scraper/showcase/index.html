Project Inspiration

Inspired by the New York Times' "Manosphere Report" — an internal tool that uses LLMs to monitor podcasts and surface editorial intelligence.

The NYT Approach

The New York Times built an AI tool that automatically downloads, transcribes, and summarizes new episodes of dozens of podcasts. Every 24 hours it generates a meta-summary with shared talking points and trends, emailed to journalists each morning.

"The Manosphere Report gave us a really fast and clear signal... There was a direct link between seeing that and then diving in to actually cover it." — Zach Seward, NYT Editorial Director for AI Initiatives

BMJ Pod Monitor adapts this approach for health, medical, and public health journalism — monitoring podcasts that discuss clinical practice, health policy, and medical research to keep BMJ's editorial team informed.

Core Features

A complete pipeline from podcast feeds to actionable editorial intelligence.

🎙

Podcast Feed Monitoring

Automatically track RSS feeds from health and medical podcasts. Add or remove podcasts, categorize them, and control which are actively monitored.

📝

AI Transcription

Audio episodes are automatically transcribed using speech-to-text models (OpenAI Whisper or AWS Transcribe), creating searchable text records.

🧠

LLM Summarization

Transcripts are processed by LLMs (Claude, GPT, or Gemini) to generate concise summaries, gists, key themes, and notable quotes.

📊

Cross-Podcast Analysis

Daily and weekly meta-analysis across all monitored podcasts identifies shared themes, emerging trends, and shifts in rhetoric.

✉

Email Digests

Professional email newsletters delivered daily and weekly to editorial distribution lists, with actionable story leads and investigation suggestions.

⚙

Admin Dashboard

Web-based SPA for managing podcasts, reviewing transcripts and summaries, previewing emails, and configuring the platform.

Scheduled Operations

Continuous Scraping

RSS feeds are polled periodically. New episodes are automatically downloaded, transcribed, and summarized as they appear.

Daily Digest (8:00 AM)

Each morning, summaries from the last 24 hours are collated into a cross-podcast analysis, identifying shared themes and notable shifts.

Weekly Analysis (Monday)

Weekly meta-analysis compiles daily digests into a broader trend report with story leads and investigation suggestions.

Try It Out

Get started with the local development environment.

Quick Start (Local Development)

# 1. Navigate to the backend
cd nyt-pod-scraper/backend

# 2. Create a virtual environment
python3 -m venv venv
source venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. (Optional) Set your LLM API key
export LLM_API_KEY=your-api-key-here
export LLM_PROVIDER=anthropic  # or openai, google

# 5. Start the server
python app.py

# 6. Open in browser
# http://localhost:5001

# 7. Click "Seed Demo Data" on the dashboard for sample content
